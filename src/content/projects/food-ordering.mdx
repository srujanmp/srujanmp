- Built a scalable food ordering platform using a microservices architecture with Go and Gin.
- Developed 7 independent services including Auth, User, Restaurant, Order, Delivery, and Notification.
- Implemented event-driven communication using RabbitMQ and transactional outbox pattern.
- Designed REST APIs with JWT authentication, Redis-based rate limiting, and role-based access control.
- Containerized services using Docker and orchestrated deployments using Kubernetes.
- Built the frontend using Next.js for restaurant browsing, order placement, and real-time order tracking.
